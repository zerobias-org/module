# DataProducer Interface V2.0 - Enhancement Project Plan

## Executive Summary

This document outlines the V2.0 enhancement plan for the DataProducer Interface, addressing critical pagination bugs and adding advanced property selection capabilities. The current implementation has fundamental flaws in cursor-based pagination support and lacks the flexibility needed for efficient data querying across diverse data sources.

## Critical Issues Identified

### 1. PagedResults Pagination Bugs (High Priority)

**Problem**: The PagedResults library has broken cursor-based pagination implementation:

- **Parameter Conflicts**: Sends both `pageToken` AND `pageNumber` in same request
- **Iterator Logic**: `asyncGenerator()` assumes page number increment, cannot work with tokens
- **Header Detection**: Incomplete pageToken response header parsing
- **Mixed Mode Support**: No automatic detection of pagination strategy

**Impact**: 
- Cursor-based APIs (DynamoDB, GraphQL, etc.) fail or return inconsistent results
- Large dataset pagination inefficient or impossible
- Memory issues with offset-based pagination on large collections

### 2. Missing Property Selection (Medium Priority)

**Problem**: Collection search returns entire objects, causing:
- Excessive network payload for large documents
- Client-side filtering overhead
- Poor performance for mobile/low-bandwidth scenarios

**Requested**: Dot notation property selection (`["email", "contact.phone", "contact.name"]`)

### 3. Container Search Limitations (Medium Priority)

**Problem**: No dedicated search operation for container objects:
- `/children` endpoint lacks filtering capabilities
- No RFC4515 filter support for child object metadata
- Cannot search container hierarchies efficiently

### Task 2.2: Rewrite AsyncIterator ⚠️ CRITICAL
**Current Broken Logic** (`PagedResults.ts:627-658`):
```typescript
// ❌ WRONG - assumes page number increment
while (!done) {
  pageNum += 1;  // Cannot work with tokens!
}
```

**Required Token-Aware Logic**:
```typescript
async* asyncGenerator() {
  if (!this.baseUrl) {
    // Just iterate current items
    yield* this.items;
    return;
  }

  let hasNextPage = true;
  let currentPage = this.items;
  let currentPageNum = this.pageNumber;

  // Yield current page first
  yield* currentPage;

  while (hasNextPage) {
    let nextPage: T[];
    
    if (this.pageToken) {
      // Cursor-based pagination
      nextPage = await this.fetchNextPageByToken();
      // pageToken updated by response headers in doFetch()
      hasNextPage = !!this.pageToken;
    } else if (this.pageCount && currentPageNum < this.pageCount) {
      // Offset-based pagination
      currentPageNum++;
      nextPage = await this.fetchPage(currentPageNum);
      hasNextPage = currentPageNum < this.pageCount;
    } else {
      hasNextPage = false;
      break;
    }

    if (nextPage.length === 0) {
      hasNextPage = false;
    } else {
      yield* nextPage;
    }
  }
}
```

### Task 2.3: Add Pagination Mode Detection
**Auto-Detection Strategy**:
```typescript
class SmartPagedResults<T> extends PagedResults<T> {
  detectPaginationMode(response: AxiosResponse): 'cursor' | 'offset' {
    const headers = response.headers;
    const body = response.data;

    // Check for cursor indicators
    if (headers['x-next-page-token'] || 
        headers['pagetoken'] ||
        headers['next-page-token'] ||
        body.pageToken ||
        body.nextPageToken) {
      return 'cursor';
    }

    // Check for offset indicators  
    if (headers['x-total-count'] || 
        headers['count'] ||
        body.pageCount ||
        body.totalCount) {
      return 'offset';
    }

    return 'offset'; // Default fallback
  }
}
```

### Task 2.4: Standardize Header Handling

private extractPageToken(response: AxiosResponse): string | undefined {

## Phase 4: Implementation Strategy (Week 3)

### Task 4.1: Property Selection Implementation
**Core Algorithm**:
```typescript
class PropertySelector {
  select(obj: any, properties: string[]): any {
    if (!properties || properties.length === 0) {
      return obj; // Return all properties
    }

    const result: any = {};
    
    for (const prop of properties) {
      if (prop.includes('.')) {
        // Nested property (e.g., "contact.phone")
        this.selectNestedProperty(obj, result, prop);
      } else {
        // Top-level property (e.g., "email")
        this.selectTopLevelProperty(obj, result, prop);
      }
    }
    
    return result;
  }

  private selectTopLevelProperty(source: any, target: any, requested: string): void {
    // Find matching field with fuzzy matching
    for (const [key, value] of Object.entries(source)) {
      if (this.matchesField(requested, key)) {
        target[key] = value;
        return;
      }
    }
  }

  private selectNestedProperty(source: any, target: any, path: string): void {
    const [parentKey, ...childPath] = path.split('.');
    const childProp = childPath.join('.');
    
    // Find parent object
    for (const [key, value] of Object.entries(source)) {
      if (this.matchesField(parentKey, key) && typeof value === 'object') {
        if (!target[key]) target[key] = {};
        
        if (childPath.length === 1) {
          // Direct child property
          this.selectTopLevelProperty(value, target[key], childProp);
        } else {
          // Nested child property  
          this.selectNestedProperty(value, target[key], childProp);
        }
        return;
      }
    }
  }

  private matchesField(requested: string, available: string): boolean {
    return this.normalizeFieldName(requested) === this.normalizeFieldName(available);
  }

  private normalizeFieldName(field: string): string {
    return field.toLowerCase()
      .replace(/[-_\s]/g, '')           // Remove separators
      .replace(/([a-z])([A-Z])/g, '$1$2'); // Handle camelCase
  }
}
```

### Task 4.3: SearchChildren Integration
**Implementation Pattern**:
```typescript
// In DataProducer implementations
async searchChildObjects(
  objectId: string,
  filter?: string,
  properties?: string[],
  scope = 'one_level',
  pageToken?: string,
  pageNumber = 1,
  pageSize = 100
): Promise<PagedResults<Object>> {
  
  // 1. Validate container object
  const container = await this.getObject(objectId);
  if (!container.objectClass.includes('container')) {
    throw new UnsupportedOperationError('Object is not a container');
  }
  
  // 2. Apply RFC4515 filter to child objects
  let children = await this.getChildrenWithFilter(objectId, filter, scope);
  
  // 3. Apply property selection
  if (properties && properties.length > 0) {
    const selector = new PropertySelector();
    children = children.map(child => selector.select(child, properties));
  }
  
  // 4. Apply pagination (smart mode detection)
  return this.paginateResults(children, pageToken, pageNumber, pageSize);
}
```
